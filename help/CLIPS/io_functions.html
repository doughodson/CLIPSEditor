IO Functions

The following functions perform I/O operations.

close

Closes a file.

(close [<logical-name>])

format

Sends formated output to the specified logical name.

(format <logical-name> <string-expression> <expression>*)

open

Opens a file.

(open <file-name> <logical-name> [<mode>])

<mode> ::= "r" | "w" | "r+" | "a" | "wb"

printout

Sends unformated output to the specified logical name.

(printout <logical-name> <expresion>*)

read

Reads a single-field value from the specified logical name.

(read [<logical-name>])

readline

Reads an entire line as a string from the specified logical name.

(readline [<logical-name>])

remove

Deletes a file.

(remove <file-name>)

rename

Changes the name of a file.

(rename <old-file-name> <new-file-name>)

get-char

Allows a single character to be retrieved from a logical name.

(get-char [<logical-name>])

read-number

Allows a user to input a single number using the localized format.

(read-number [<logical-name>])

set-locale

Allows a locale to be specified for the numeric format behavior of the format and read-number functions.

(set-locale [<locale-string>])
